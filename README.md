# Mumtazzam-Soal-Mudah-1
Alur kerja program :
1. Program menampilkan "Selamat datang di layanan ekspedisi kami!
2. Pengguna memasukkan nilai berat barang
3. Selanjutnya, pengguna memasukkan jarak tempuh barang dalam km
4. Setelah itu, pengguna lanjut memasukkan panjang, lebar, tinggi barang
5. Selanjutnya, program mengecek jika jarak kurang dari atau sama dengan 10 km, maka berat barang dikali dengan Rp. 4250
6. Jika jarak lebih dari 10 km, maka value berat yang di inputkan oleh user akan dikali dengan Rp. 6000
7. Selanjutnya, hitung volume dengan mangalikan panjang, lebar, tinggi yang sudah di inputkan olah user sebelumnya
8. Jika volume lebih dari 100 cm3, maka fee akan bertambah Rp. 50000
9. Selanjutnya, program menampilkan hasil dari program yakni "Total biaya pengiriman : Rp. hasil"
